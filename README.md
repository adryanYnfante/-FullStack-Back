# Reto para SofkaU

- Sistema de preguntas y respuestas, donde el usuario registrado pueden realizar preguntas.
- Los usuarios que no están logueados no pueden ingresar al panel principal.
- Sólo los usuarios registrados podrán responder a las preguntas que estén en la plataforma.

# Tecnologías

JAVA | SPRING BOOT | JUnit | Mockito | Swagger
